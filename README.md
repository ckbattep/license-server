# CitoLaw License Server v2

> Серверная подсистема валидации лицензий для Android-приложения.

## ✅ Что реализовано:
- Один источник истины (сервер)
- HMAC-подпись токенов (без JWT)
- Device binding
- Отзыв токенов (`/license/revoke`)
- Поддержка TRIAL + ACTIVE

## 🔑 Ключи

- Приватный ключ: `keys/ed25519_private.key`
- Публичный ключ: `keys/ed25519_public.key`

⚠️ Оба файла **обязательны** для запуска сервера.
